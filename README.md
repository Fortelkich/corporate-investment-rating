# Прогнозирование кредитного рейтинга компаний

Проект посвящен построению моделей машинного обучения для прогнозирования укрупненных групп кредитного риска компаний на основе финансовых показателей.

## Цель проекта

Цель — разработать ML-пайплайн для прогнозирования кредитного рейтинга компаний и определить финансовые признаки, наиболее связанные с уровнем кредитного риска.

Дополнительно рассматривается дополнительная задача: прогнозирование показателя `operatingCashFlowSalesRatio`, который связан с кредитным рейтингом и отражает способность компании генерировать операционный денежный поток относительно выручки.
